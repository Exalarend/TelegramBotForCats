# Системные картинки

Положите сюда картинки для системных уведомлений.

Используемые по умолчанию пути (см. `config/system_notifications.yaml`):

- `assets/system/Garfield_1.jpeg`
- `assets/system/Garfield_2.png`
- `assets/system/Garfield_3.png`
- `assets/system/Friday_1.jpg`

## Как выбирается контент

- Выбирается **одна** картинка из списка по весам (`weight`).
- Затем выбирается **текст из набора этой картинки** (`texts`) тоже по весам.

Если файла по пути нет — бот просто не будет выбирать этот вариант.
